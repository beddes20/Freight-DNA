1. AM assigning a task to an LM
2. Any user assigning a task to themselves

Example:
- an AM should be able to assign an LM a task to source carriers for a lane we won on an award
- a rep should be able to send themselves a reminder or follow-up task
- a manager/director/NAM should still be able to assign tasks to reps or LMs
