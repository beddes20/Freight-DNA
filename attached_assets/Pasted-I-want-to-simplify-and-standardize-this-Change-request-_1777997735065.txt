explicit quote/opportunity owner if already assigned,

exact customer-linked shared/distribution email match if supported,

This should be a fallback/default ownership rule, not a forced overwrite.

If a quote is already explicitly assigned, keep that assignment.

If the system matches a known account but no explicit rep is present, use Account Owner instead of leaving Rep blank.

This is important:
if a customer emails from a distribution/shared address and that email is associated to the customer account, the system should still fall back to the Account Owner.

In other words:

account match first,
